# 🥗 Sobre o Projeto

**FitTrack Nutrition** é uma aplicação web dedicada ao planeamento e acompanhamento nutricional para pessoas que procuram melhorar a sua composição corporal.

O projeto foi desenvolvido para organizar:

✅ alimentação diária  
✅ refeições  
✅ controlo nutricional  
✅ objetivos físicos  
✅ acompanhamento de progresso  

Com foco em:

🔥 perda de gordura  
💪 manutenção e ganho de massa muscular  
⚡ melhoria da performance nos treinos  

---

# 🎯 Objetivo

Criar uma plataforma simples e visual onde seja possível acompanhar uma estratégia nutricional completa alinhada com um plano de treino.

O objetivo principal:

> Transformar a nutrição numa ferramenta prática para alcançar resultados sustentáveis.
